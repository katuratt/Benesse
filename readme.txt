pythonの仮想環境下で，
$ streamlit run app.py
を実行することで，アプリは起動します．
起動しない場合は，__pycache__を消してください．__pycache__はkatsuraの中とappsの中にございます．

念の為，必要となるモジュールはrequirements.txtに示しましたが，漏れがある場合がございます．

こだわった点は以下の通りです．
①個人情報の収集許諾やインターンシップに応募する際の個人情報の記入等の，些細なところまで再現した点．
②「本質的な評価軸」が最適化され，自分にあった企業の範囲が狭まっていくところをしっかりと実装した点．
③「自己分析」のページでは，他のページに移動した後に戻っても，記入した情報は入ったままですが，「会社体験」のタブでは，インターンシップに応募したり，ゲーム後のアンケートに答えた後は，初期ページに戻るようにした点．
④カテゴライズのページで，特定の産業のみをグラフ中に表示させたり，会社名検索ができるようにした点です．対象となる宮崎タクヤ君は現在は使わないと思いますが，就活が進むにつれ，検索機能は欲しいであろうと考え，実装しました．
